# CarNow API

# To run locally
1. Clone project
2. Set environment variables in .env file
3. Run docker
4. <a href="https://documenter.getpostman.com/view/20768087/2s8YemvuVd" target="blank">Api Docs</a>
```
docker-compose up -d
```

# To run tests
1. Clone project
2. ```npm install```
3. ```npm run test```
